# Product Landing Page

A responsive product landing page built with **pure HTML and CSS** as part of freeCodeCamp's Responsive Web Design certification.

## Live Demo
Check out the live version here:  
[live page link will be here](https://github.com/Rustom-yadav/product-landing-page.git)

## Features
- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Hero section with call-to-action
- Feature cards with icons
- Pricing section
- Contact form
- Clean and modern UI

## Technologies Used
- HTML5
- CSS3
